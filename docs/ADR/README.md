# ADR Index

Architecture Decision Records (ADRs) capture important technical decisions and their context.

## Rules

1. One decision per ADR file.
2. ADRs are immutable after acceptance; supersede with a new ADR.
3. Every significant architecture/security/infrastructure change must reference an ADR.

## Status Values

- `Proposed`
- `Accepted`
- `Deprecated`
- `Superseded by ADR-XXXX`
